Chapter 3: Computer Softeware Operating System

Introduction

An operating system (OS) is an integrated set of programs that control overall resources such as CPU, memory, and input-output devices of the computer system. The major objective of an operating system is to improve the performance and efficiency of a computer system. Like the manager of a company, an operating system is responsible for the smooth and efficient functioning of the entire computer system. The operating system provides the platform for other application programs/software to run and execute. It provides the user with an interface so that the user can easily communicate with computers, which is more convenient to use and operate.

Functions of Operating System

  • Controls, monitors, and coordinates the overall operation of our computer system.
  • Acts as an interface between the user and the computer.
  • Provides a platform to develop, run, and execute other computer programs.
  • Manages hardware resources such as CPU, memory, input-output terminals, networking equipment, etc.
  • Hides programming and hardware complexity from the user.

Types of Operating System

A) Types of Operating Systems Based on Processing

  • Multi-programming OS: Executes multiple user programs on a single processor, improving resource utilization. Examples: multi-tasking, multi-processing, and multi-user systems.
  • Multi-tasking OS: Allows multiple tasks to run concurrently in a single-user environment. Examples: MS-Windows, Linux, macOS.
  • Multiprocessing OS: Uses more than one processor to run programs simultaneously, improving execution speed. Example: UNIX.
  • Time-Sharing OS: Allows multiple users to share CPU time. Tasks switch frequently, providing an immediate response. Examples: MS-DOS, UNIX.
  • Batch Processing OS: Groups similar jobs into batches for sequential processing without user interaction during execution. Common in payroll, weather forecasting, etc.
  • Multi-threading OS: Divides a program into threads that can be executed concurrently, improving efficiency.
  • Real-time OS: Processes tasks in a predefined time, suitable for critical operations like flight seat availability and rocket launching systems.
  • Online Processing OS: Processes transactions immediately as they occur, allowing user interaction during processing.

B) Types of Operating Systems Based on User Interface (UI)

  • Character User Interface (CUI): Operates using text-based commands. Less user-friendly, requiring users to memorize commands. Examples: MS-DOS, UNIX.
  • Graphical User Interface (GUI): Allows interaction using graphics, icons, and dialogue boxes. More user-friendly and easier for non-technical users.

Software

Software is the intangible set of instructions, data, or programs that enable computers and other devices to perform specific tasks. It encompasses a wide range of applications, from operating systems that manage hardware resources to specialized programs designed for tasks like word processing, image editing, or web browsing. Software development involves a systematic process of designing, coding, testing, and deploying these programs, often requiring collaboration among developers, designers, and testers. The quality and efficiency of software are crucial for its usability and reliability, impacting everything from personal productivity to complex industrial operations. Updates and maintenance are essential for addressing bugs, enhancing features, and ensuring compatibility with evolving hardware and software environments.

Chapter 4: Application Packages

4.1 Introduction

Application packages are software suites designed to help users perform specific tasks efficiently. Among the most widely used applications are Microsoft Word, Excel, and PowerPoint, which are part of the Microsoft Office suite. Microsoft Word is a word processing program that allows users to create, edit, and format text documents, making it ideal for writing reports, letters, and essays. Excel is a powerful spreadsheet application that enables users to organize, analyze, and visualize data through tables, charts, and formulas, making it essential for tasks involving calculations and data management. PowerPoint is a presentation software that allows users to create visually engaging slideshows, incorporating text, images, and multimedia elements to effectively communicate ideas. Together, these applications enhance productivity and facilitate communication in both personal and professional settings.

4.2 Microsoft Word

Microsoft Word is a word processing application that allows users to create, edit, and print documents. It is widely used for creating letters, reports, resumes, and other types of documents.

  • Features: Word processing, formatting, editing, and printing documents.
  • Tools: Spell checker, grammar checker, thesaurus, and dictionary.
  • File Formats: .docx, .doc, .rtf, and .txt.

4.3 Microsoft Excel

Microsoft Excel is a spreadsheet application that allows users to create, edit, and analyze data in a tabular format. It is widely used for creating budgets, charts, and graphs.

  • Features: Data entry, formatting, and analysis, as well as creating charts and graphs.
  • Tools: Formulas, functions, and pivot tables.
  • File Formats: .xlsx, .xls, .csv, and .txt.

4.4 Microsoft PowerPoint

Microsoft PowerPoint is a presentation application that allows users to create, edit, and display presentations. It is widely used for creating slideshows, presentations, and lectures.

  • Features: Creating and editing slides, adding text, images, and multimedia, and presenting slideshows.
  • Tools: Templates, themes, and transitions.
  • File Formats: .pptx, .ppt, .pps, and .pdf.

4.5 Other Microsoft Office Applications

In addition to Word, Excel, and PowerPoint, Microsoft Office includes several other applications, such as:

  • Outlook: An email client that allows users to send and receive emails.
  • Access: A database management application that allows users to create and manage databases.
  • Publisher: A desktop publishing application that allows users to create and design publications.
